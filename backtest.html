Methodologie

Source des donnees : Yahoo Finance API v8 Chart - donnees ajustees en cloture (2 ans, journalier).

Strategie A - Simple :

  • Signal d'achat : MA20 croise au-dessus de MA50 (croisement doree)
  • Signal de vente : MA20 croise en-dessous de MA50, OU stop-loss fixe -2%, OU take-profit fixe +4%
  • Capital : $10 000 par titre, investi integralement a chaque signal
  • Horizon : 2 ans de donnees, 5 titres (AAPL, MSFT, TSLA, NVDA, AMZN)

Strategie B - Filtree :

  • Memes signaux MA20/MA50 + filtres additionnels : RSI(14) entre 30-70, volume superieur a la moyenne 20 jours
  • Persistance 2 jours : le croisement doit etre confirme 2 jours consecutifs avant entree
  • Stop-loss dynamique trailing : ATR(14) x 1.5 (remonte automatiquement avec le prix)
  • Portefeuille : $500 total, maximum $100 par position, maximum 3 positions simultanees
  • Circuit breaker : toute activite suspendue si le drawdown du portefeuille depasse -10%

Metriques calculees :

  • Rendement total : (Capital final - Capital initial) / Capital initial x 100
  • Ratio de Sharpe annualise : (Moyenne des rendements par trade / Ecart-type) x sqrt(trades/an), taux sans risque = 0
  • Drawdown maximum : pire baisse depuis un pic d'equite
  • Taux de gain : % de trades clotures en profit

Avertissement : Ce backtest est fourni a titre informatif uniquement. Les performances passees ne garantissent pas les resultats futurs. Aucun frais de courtage ni impact de marche n'est inclus dans cette simulation.
